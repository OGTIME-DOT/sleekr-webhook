# Sleekr Webhook

Webhook service for syncing ActiveCampaign contact updates to Airtable.

## Purpose
- Capture user Zone + Interest from ActiveCampaign
- Update corresponding records in Airtable
- Support Sleekr beta user qualification and rollout

## Stack
- Vercel (serverless function)
- ActiveCampaign (trigger)
- Airtable (database)
